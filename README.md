# Smart Study Assistant

## Project Overview

The Smart Study Assistant is a Python-based chatbot that answers basic programming questions entered by the user. The chatbot uses a predefined knowledge base stored in a Python dictionary and keeps a record of all conversations in a text file. This project was developed to practise Python programming concepts such as dictionaries, loops, file handling, and exception handling.

## Features

* Menu-driven interface
* Answer basic programming questions
* Uses a dictionary as a knowledge base
* Saves conversation history to a text file
* Displays previous chat history
* Handles invalid menu input using exception handling
* Converts user input to lowercase for easier matching

## Technologies Used

* Python
* Dictionaries
* Loops
* Conditional Statements
* File Handling
* Exception Handling

## How the Program Works

1. The program displays a menu with different options.
2. The user can ask a programming-related question.
3. The chatbot searches for the question in its dictionary.
4. If the question exists, the correct answer is displayed.
5. If the question is not available, the chatbot displays an appropriate message.
6. Every conversation is saved in a text file called **chat_history.txt**.
7. The user can view previous conversations from the menu.
